## 关于 CK

CK 是一个面向跨境电商与短视频内容生成的工具集合（包括海外仓 XML 报文生成、Excel 数据提取、AI 视频/分镜生成等），目标是帮助卖家和运营人员提高数据处理与内容产出效率。

## 快速上手

- 仓库地址：https://github.com/ludemandasi/CK
- 运行本地静态页面：
  1. 克隆仓库：
     ```bash
     git clone https://github.com/ludemandasi/CK.git
     cd CK
     ```
  2. 如果项目是静态页面，直接打开 `index.html`；或使用本地服务器：
     ```bash
     npx serve .
     # 打开 http://localhost:3000
     ```

- 常见入口：
  - 跨境报文生成器（生成 CEB303/XML）
  - 单一窗口数据提取（上传 Excel 并导出所需字段）
  - AI 视频 / 分镜生成（上传图片、选择模型并生成短视频）

## 如何参与

- 提交 Bug：使用仓库的 Issue -> Bug report 模板提供必要信息（步骤、复现、环境）。
- 提交功能建议：使用 Feature request 模板。
- 贡献代码：阅读 CONTRIBUTING.md，并创建基于 `main` 的分支提交 PR。
