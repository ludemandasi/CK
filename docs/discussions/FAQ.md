# 常见问题（FAQ）

这篇 FAQ 用于收集用户常见问题与解答，方便新人快速找到解决办法。

## Q1：如何本地运行项目？
A：通常为静态页面，直接打开 `index.html` 即可；如果需要本地服务器（推荐）：
```bash
npx serve .
# 打开 http://localhost:3000
```

## Q2：如何生成 XML 报文？
A：进入 “CEB303 跨境电商海外仓订单报文生成器” 页面，填入订单信息或上传 Excel，点击 “预览 XML” 查看生成的 XML，确认后可下载 `.xml` 文件。

## Q3：我上传 Excel 后有字段缺失怎么办？
A：请检查上传的 Excel 是否为“单一窗口导出”格式，或在左侧字段选择器中勾选需要导出的字段。如仍有问题，请在 Issue 中附上（脱敏）示例文件或截图。

## Q4：AI 视频生成需要配置哪些参数？
A：通常需要选择：
- 模型（例如 gpt-4o 等）
- 参考图片或参考视频（可选，缩短生成时间）
- 视频比例与时长
- 是否生成旁白（语音）

## Q5：如何配置 API Key？
A：在分镜 Agent 或 AI 视频生成页面的“模型配置”区域填写 API Key 与 Base URL（仅项目维护者或信任用户应提供），请不要在公共讨论中贴出明文 Key。

## Q6：我想贡献代码，有模板吗？
A：请参阅仓库根目录的 CONTRIBUTING.md，按照分支/提交规范创建 PR。请在 PR 描述中说明变更内容与测试步骤。
