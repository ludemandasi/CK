# Roadmap / 功能想法

欢迎在此帖中讨论未来功能、优化方向和优先级排序。以下为当前的候选项，欢迎投票或补充：

## 当前候选功能

1. 核心：增强 Excel 自动映射支持（更多导出格式，自动识别字段名别名）。
2. 用户体验：在报文生成器中加入更多校验规则（金额校验、税号格式等）。
3. AI：接入更多视觉/语音模型，支持多语言旁白与字幕生成。 
4. CI：添加自动化测试与构建 Pipeline（GitHub Actions）。
5. 文档：补全使用示例与视频教学，增加快速入门视频。
6. 社区：启用 Discussions 并建立分类（Q&A/Show & Tell/Ideas/Announcements）。

## 如何参与 Roadmap 讨论

- 如果你赞成某项功能，请在该项下回复“+1”并说明你的使用场景。 
- 如果你愿意贡献实现，请在回复中说明你的技术栈与计划（例如：我可以负责添加 GitHub Actions，使用 Node.js 构建脚本）。
