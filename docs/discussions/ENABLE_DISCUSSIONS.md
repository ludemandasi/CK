# 如何启用 GitHub Discussions（说明）

要在 GitHub 仓库启用 Discussions，请按以下步骤操作（需要仓库管理员权限）：

1. 打开仓库页面：https://github.com/ludemandasi/CK
2. 点击页面上方的 "Settings"（设置） 标签。 
3. 在左侧栏选择 "Features" 或在 Settings 页面中找到 "Features" 区块。
4. 勾选 "Discussions"（启用 Discussions）。
5. 保存设置。启用后，页面上方会出现 "Discussions" 选项卡。

创建分类（Categories）：
1. 进入仓库的 Discussions 页面，点击 "New category"（新建分类）。
2. 建议创建以下分类：
   - Q&A（问答/使用问题）
   - Show & Tell（演示/用户案例）
   - Ideas（功能/路线图）
   - Announcements（公告/发布）

发布首帖：
1. 选择合适分类（例如 Announcements 或 Q&A），点击 "Start a new discussion"。
2. 将 docs/discussions/Welcome.md 的内容复制粘贴到文本框，设置标题为 "Welcome to CK — 项目简介与如何参与"，发布。
3. 同理发布 FAQ 与 Roadmap。

安全提示：
- 不要在 Discussions 或 Issues 中贴出明文 API Key、密码或敏感信息；如果需要排查问题，请提供脱敏数据或截图。
